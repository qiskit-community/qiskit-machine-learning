Qiskit Machine Learning API Reference#

Qiskit Machine Learning module (qiskit_machine_learning)#

Qiskit Machine Learning is an ML framework that comes with essential tools like quantum kernels and quantum neural networks. These tools are the building blocks of the framework and can be used for building and training classification, regression, and other models.

QiskitMachineLearningError

Class for errors returned by Qiskit Machine Learning module.
